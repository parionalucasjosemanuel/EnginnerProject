🟢 Easy Mode

5 Questions

1. Which device connects multiple computers inside a LAN?

✅ Correct Answer: Switch
A switch allows devices inside a Local Area Network to communicate efficiently.

2. Which cable is commonly used between a PC and a Switch?

✅ Correct Answer: Straight-through cable
Straight-through cables connect different device types like PC to Switch.

3. Which layer of the OSI model handles IP addresses?

✅ Correct Answer: Network Layer
The Network Layer handles logical addressing and routing using IP.

4. Which protocol is used to browse websites?

✅ Correct Answer: HTTP
HTTP is the protocol browsers use to access websites.

5. What does LAN stand for?

✅ Correct Answer: Local Area Network
LAN means Local Area Network and connects devices in a limited area.

🟡 Medium Mode

5 Questions

6. Which protocol dynamically assigns IP addresses?

✅ Correct Answer: DHCP
DHCP automatically provides IP addresses to devices.

7. What command checks connectivity between two devices?

✅ Correct Answer: ping
Ping sends ICMP packets to verify network connectivity.

8. Which device routes traffic between networks?

✅ Correct Answer: Router
Routers connect different networks and forward packets.

9. Which topology uses a central switch?

✅ Correct Answer: Star
Star topology connects all devices to one central device.

10. Which protocol translates domain names into IP addresses?

✅ Correct Answer: DNS
DNS converts readable domain names into IP addresses.

🔴 Hard Mode

5 Questions

11. Which routing protocol uses hop count as metric?

✅ Correct Answer: RIP
RIP measures routes using hop count.

12. What is the default port for HTTPS?

✅ Correct Answer: 443
HTTPS encrypts web traffic using port 443.

13. Which command displays router interfaces?

✅ Correct Answer: show ip interface brief
That command shows interface status and IP addresses.

14. Which protocol secures remote CLI access?

✅ Correct Answer: SSH
SSH encrypts remote access communications securely.

15. Which subnet mask belongs to a /24 network?

✅ Correct Answer: 255.255.255.0
A /24 network contains 24 bits for the network portion.
